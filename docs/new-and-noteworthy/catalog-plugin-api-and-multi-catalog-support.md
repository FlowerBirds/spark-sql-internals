* [Developing CatalogPlugin](../demo/developing-catalogplugin.md)

## Example

```sql
SHOW NAMESPACES;

SHOW CURRENT NAMESPACE;

CREATE NAMESPACE IF NOT EXISTS my_ns;

USE NAMESPACE my_ns;

SHOW CURRENT NAMESPACE;
```
